# simple-form

##### Add this in your project's build.gradle (project level)
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

##### Add this in your project's build.gradle (app level)
```
	dependencies {
	        implementation 'com.github.lspradeep:simple-form:v1.0.0'
	}
```
